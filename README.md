Bennett Foods
Bennett Bites Hub is a modern web application built with TypeScript, Vite, and Tailwind CSS. It serves as a platform to showcase and manage food-related content, possibly for a food blog or a culinary business.​

🚀 Features
Fast and efficient development with Vite

Type-safe codebase using TypeScript

Responsive and customizable UI with Tailwind CSS

Organized project structure with src/ and public/ directories

Linting configured via eslint.config.js

PostCSS support for extended CSS functionalities​
